What is the difference between `null` and `undefined`?

4.What is the significance of `async`/`await` in fetching API data?
#async দিয়ে কোনো ফাংশনকে asynchronous করা হয়, মানে সেই ফাংশন সবসময় একটি Promise রিটার্ন করবে।

#await দিয়ে আমরা Promise resolve না হওয়া পর্যন্ত অপেক্ষা করি, তারপর ফলাফল পাই।
5.Explain the concept of Scope in JavaScript (Global, Function, Block).
# Global মানে সব জায়গায় পাওয়া যাবে, Function মানে শুধু ফাংশনের ভেতরে, আর Block মানে {} এর ভেতরে সীমাবদ্ধ।
